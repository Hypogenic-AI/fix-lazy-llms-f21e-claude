# Code Repositories

Cloned repositories for the "Fixing Lazy LLMs" research project.

## Repositories

### self-refine
- **Paper**: Self-Refine: Iterative Refinement with Self-Feedback (Madaan et al., 2023)
- **URL**: https://github.com/madaan/self-refine
- **Relevance**: Core self-refinement framework; demonstrates iterative LLM self-critique
- **Key files**: src/ contains task-specific prompts for generation, feedback, and refinement

### TALE
- **Paper**: Token-Budget-Aware LLM Reasoning (Han et al., 2024)
- **URL**: https://github.com/GeniusHTX/TALE
- **Relevance**: Token budget control for reasoning; TALE-EP (training-free) and TALE-PT methods

### reflexion
- **Paper**: Reflexion: Language Agents with Verbal Reinforcement Learning (Shinn et al., 2023)
- **URL**: https://github.com/noahshinn/reflexion
- **Relevance**: Self-reflection framework using verbal feedback for iterative improvement
